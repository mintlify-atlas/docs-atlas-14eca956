### Detection System

The AI system detects six categories of hygiene issues:

**Allowed Issues** (from `app/vision.py:65-70`):
```python
# Allowed issues:
- wet_seat         # Water on toilet seat
- pee_stain        # Urine stains
- feces_stain      # Fecal matter
- toilet_paper_floor  # Paper on floor
- trash_on_floor   # Other debris
- water_puddle     # Water on floor
```

**Scoring Rubric** (from `app/vision.py:74-85`):
```python
Start from a base score of 100.

Subtract points as follows:
- feces_stain: −35 points each
- pee_stain: −20 points each
- wet_seat: −10 points each
- water_puddle: −10 points each
- toilet_paper_floor: −5 points each
- trash_on_floor: −5 points each

Rules:
- Multiple instances of the same issue stack.
- Clamp the final score between 0 and 100.
- If no issues are detected, score must be 100.
```

### Bounding Box Annotations

Detected issues are highlighted with bounding boxes on the image.

**Drawing Boxes** (from `app/vision.py:265-303`):
```python
def draw_boxes(image_bytes: bytes, detections: list) -> bytes:
    """Draw bounding boxes on the image for detected issues"""
    img = cv2.imdecode(
        np.frombuffer(image_bytes, np.uint8),
        cv2.IMREAD_COLOR
    )

    h, w = img.shape[:2]
    for d in detections:
        x1, y1, x2, y2 = d["bbox"]

        # Clamp to image bounds
        x1 = max(0, min(x1, w))
        x2 = max(0, min(x2, w))
        y1 = max(0, min(y1, h))
        y2 = max(0, min(y2, h))
        
        label = f'{d["issue"]} {d.get("confidence", 0):.2f}'
        
        cv2.rectangle(img, (x1, y1), (x2, y2), (0, 0, 255), 2)
        cv2.putText(
            img, label, (x1, max(15, y1 - 5)),
            cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 0, 255), 1
        )

    _, encoded = cv2.imencode(".jpg", img)
    return encoded.tobytes()
```

<Note>
  Annotated images are automatically uploaded to Supabase Storage and accessible via public URLs.
</Note>

## Gamification System

Users earn points and progress through tiers based on their contributions to cleanliness.

### Score Delta System

Your personal score changes based on toilet cleanliness at inspection time.

**Calculation** (from `app/main.py:228-239`):
```python
def calculate_score_delta(cleanliness_score: int) -> int:
    """Calculate score delta based on cleanliness score"""
    if cleanliness_score >= 90:
        return 5  # Bonus for very clean
    elif cleanliness_score >= 70:
        return 0  # Neutral
    elif cleanliness_score >= 50:
        return -5  # Small penalty
    elif cleanliness_score >= 30:
        return -10  # Medium penalty
    else:
        return -15  # Large penalty for very dirty
```

<Warning>
  Inspecting very dirty toilets (score < 30) results in a -15 point penalty to your personal score.
</Warning>

### Tier System

Tiers unlock access to different facility levels.

**Tier Calculation** (from `app/main.py:241-259`):
```python
def get_user_tier(score: int) -> int:
    """Calculate user tier based on score
    Returns: 1 (Bronze), 2 (Silver), or 3 (Gold)
    """
    if score >= 80:
        return 3  # Gold
    elif score >= 50:
        return 2  # Silver
    else:
        return 1  # Bronze

def get_tier_name(tier: int) -> str:
    """Get tier name from tier number"""
    tier_names = {
        1: "Bronze Standard",
        2: "Silver Standard",
        3: "Gold Standard"
    }
    return tier_names.get(tier, "Unknown")
```

| Tier | Score Range | Name | Benefits |
|------|-------------|------|----------|
| 1 | 0-49 | Bronze Standard | Basic facility access |
| 2 | 50-79 | Silver Standard | Standard facilities, basic rewards |
| 3 | 80-100 | Gold Standard | Premium facilities, leaderboard recognition |

## Nearby Toilets Map

Find and compare cleanliness scores across all facilities.

**Backend Route** (from `app/main.py:591-625`):

@app.get("/nearby-toilets")
def nearby_toilets(request: Request):
    user = get_current_user(request)
    if not user:
        return RedirectResponse(url="/login")
    
    # Fetch all toilets with their latest cleanliness scores
    toilets_res = supabase.table("toilets")
        .select("*")
        .order("name")
        .execute()
    toilets = toilets_res.data or []
    
    # For each toilet, get the latest inspection
    for toilet in toilets:
        latest_visit_res = (
            supabase.table("toilet_visits")
            .select("cleanliness_score, created_at, image_url")
            .eq("toilet_id", toilet["id"])
            .order("created_at", desc=True)
            .limit(1)
            .execute()
        )
        
        if latest_visit_res.data:
            toilet["latest_inspection"] = latest_visit_res.data[0]
        else:
            toilet["latest_inspection"] = None
    
    return templates.TemplateResponse(
        "user/nearby_toilets.html",
        {"request": request, "user": user, "toilets": toilets}
    )
```

async def upload_image_to_supabase(image_bytes: bytes, filename: str) -> str:
    """Upload image to Supabase storage and return public URL"""
    try:
        # Use admin client to bypass RLS policies
        result = supabase_admin.storage.from_("uploads").upload(
            filename, 
            image_bytes,
            file_options={
                "content-type": "image/jpeg",
                "cache-control": "3600",
                "upsert": "true"  # Overwrite if exists
            }
        )
        
        if result:
            # Get the public URL
            public_url = supabase_admin.storage.from_("uploads").get_public_url(filename)
            return public_url
        else:
            raise Exception("Failed to upload image to storage")
            
    except Exception as e:
        # Handle duplicate file errors
        if "already exists" in str(e).lower():
            supabase_admin.storage.from_("uploads").remove([filename])
            # Retry upload
            result = supabase_admin.storage.from_("uploads").upload(
                filename, image_bytes,
                file_options={"content-type": "image/jpeg"}
            )
            public_url = supabase_admin.storage.from_("uploads").get_public_url(filename)
            return public_url
        raise e
```

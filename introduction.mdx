## What is FlushLa?

FlushLla transforms toilet cleanliness monitoring into an engaging, automated experience. Using advanced AI vision analysis, the system detects hygiene issues, scores facilities, and rewards users for maintaining clean environments.

## How It Works

<Steps>
  <Step title="User Registration">
    Create your account and receive an Apple Wallet pass via email. Your pass contains a unique QR code for authentication.
  </Step>
  <Step title="Scan QR Code">
    At a facility entrance, scan your Apple Wallet pass QR code to authenticate and verify your tier access level.
  </Step>
  <Step title="Capture Inspection">
    Take a photo of the toilet using the real-time camera interface. The system triggers WebSocket-based capture.
  </Step>
  <Step title="AI Analysis">
    The image is analyzed by AI vision models (GPT-4, Cloud Vision, or Gemini) to detect hygiene issues like wet seats, stains, trash, and puddles.
  </Step>
  <Step title="Receive Score">
    Get instant cleanliness scores (0-100) with annotated images showing detected issues. Your personal points update based on the result.
  </Step>
  <Step title="Tier Progression">
    Accumulate points to advance through tiers and unlock access to premium facilities.
  </Step>
</Steps>

## Technical Stack

- **Backend**: FastAPI (Python) with async WebSocket support
- **AI Vision**: OpenAI GPT-4, Google Cloud Vision, Vertex AI Gemini
- **Database**: Supabase (PostgreSQL with Row Level Security)
- **Authentication**: Supabase Auth with session management
- **Storage**: Supabase Storage for annotated inspection images
- **Apple Wallet**: PassCreator API for dynamic pass generation
- **Real-time**: WebSocket connections for camera control
- **Image Processing**: OpenCV for bounding box annotations

## Overview

FlushLa implements a tier-based access control system where users must maintain a certain cleanliness score to access higher-tier toilets.

## Toilet Tiers

| Tier | Name | Description | Required User Score |
|------|------|-------------|---------------------|
| 1 | Bronze Standard | Basic public toilets | No restriction (score 0-49) |
| 2 | Silver Standard | Well-maintained facilities | User score ≥ 50 |
| 3 | Gold Standard | Premium, luxury toilets | User score ≥ 80 |

## Tier Calculation Logic

**Location**: `app/main.py:241`

```python
def get_user_tier(score: int) -> int:
    """Calculate user tier based on score
    Returns: 1 (Bronze), 2 (Silver), or 3 (Gold)
    """
    if score >= 80:
        return 3  # Gold
    elif score >= 50:
        return 2  # Silver
    else:
        return 1  # Bronze
```

```python
def get_tier_name(tier: int) -> str:
    """Get tier name from tier number"""
    tier_names = {
        1: "Bronze Standard",
        2: "Silver Standard",
        3: "Gold Standard"
    }
    return tier_names.get(tier, "Unknown")
```

## Access Workflow

1. **User arrives at toilet** with their digital pass (Apple Wallet / Google Pay)
2. **Toilet scanner** reads QR code from pass
3. **Scanner sends** `qr_code_data` and `toilet_id` to `/api/verify-access`
4. **System validates**:
   - Pass exists in database
   - User profile is found
   - Toilet exists
   - User tier ≥ toilet tier
5. **Access granted** if all checks pass
6. **User is logged in** automatically via session
7. **User can proceed** to use the toilet

## Improving Your Tier

Users can increase their tier by:

1. Performing toilet inspections via [Start Inspection](/api/start-inspection)
2. Keeping toilets clean (score ≥ 90 for bonus points)
3. Accumulating positive score deltas

### Score Requirements

- **Bronze → Silver**: Reach 50 points
- **Silver → Gold**: Reach 80 points

## Toilet Database Schema

Toilets are stored in the `toilets` table:

```sql
CREATE TABLE toilets (
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  location TEXT,
  current_tier INTEGER DEFAULT 1,
  last_cleanliness_score INTEGER,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

# Quick Start Guide

This guide will walk you through registering, scanning your first QR code, and performing a toilet inspection with FlushLa.

## Prerequisites

- Email address for account creation
- iPhone with Apple Wallet (for pass integration)
- Camera access for QR scanning and photo capture

## Steps

<Steps>
  <Step title="User Registration">
    ### Create Your Account

    Navigate to the registration page and provide your details. FlushLa will create both a Supabase authentication user and an Apple Wallet pass.

    **Registration Form**:
    ```html

      <input type="email" id="email" placeholder="email@example.com" required />
      <button type="submit">Create Account</button>
    </form>
    ```

    **Backend Process** (from `app/main.py:1056`):
    ```python

    auth_user = supabase_admin.auth.admin.create_user({
        "email": payload.emailRecipient,
        "password": "1234",
        "email_confirm": True
    })

    # Insert into public.users table
    supabase_admin.table("users").insert({
        "id": user_id,
        "username": payload.firstname,
        "email": payload.emailRecipient,

        "is_admin": False
    }).execute()

    # Create Apple Wallet pass via PassCreator API
    response = requests.post(
        "https://app.passcreator.com/api/v3/pass",

        "pass_identifier": pass_id
    ```

  <Step title="QR Code Authentication">
    ### Scan Your Pass

    At a facility entrance, open the QR scanner page. The system uses `html5-qrcode` library to scan your Apple Wallet pass.

    **Scanner Implementation** (from `app/templates/qr_access_scanner.html:352`):
    ```javascript
    const scanner = new Html5Qrcode("reader");

    scanner.start(
      { facingMode: "environment" },
      { fps: 10, qrbox: { width: 250, height: 250 } },
      async (decodedText) => {
        scanner.pause();
        
        // Verify access with backend
        const res = await fetch("/api/verify-access", {
          method: "POST",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify({ 
            qr_code_data: decodedText.trim(), 
            toilet_id: 1 
          })
        });

        const data = await res.json();
        
        if (data.access === "granted") {
          // Show user info and inspection interface
          showUserInfo(data.user);
        }
      }
    );
    ```

    **Backend Verification** (from `app/main.py:911`):
    ```python
    @app.post("/api/verify-access")
    def verify_access(
        request: Request,
        qr_code_data: str = Body(...),
        toilet_id: int = Body(default=1)
    ):
        # Look up pass by QR code value
        res = supabase.table("passes").select("*, users(*)")
            .eq("pass_identifier", qr_code_data).execute()
        
        user_data = res.data[0].get("users")
        
        # Calculate user's tier based on score
        user_score = user_data.get("score", 0)
        user_tier = get_user_tier(user_score)  # 1=Bronze, 2=Silver, 3=Gold
        
        # Check toilet tier requirements
        toilet_data = supabase.table("toilets")
            .select("*").eq("id", toilet_id).single().execute().data
        toilet_tier = toilet_data.get("current_tier", 1)
        
        # Verify tier access
        if user_tier < toilet_tier:
            return {
                "access": "denied",
                "reason": f"Insufficient tier. You need {toilet_tier - user_tier} more tier level(s).",
                "user_tier": user_tier,
                "required_tier": toilet_tier
            }
        
        # Grant access and set session
        request.session["user_id"] = user_data["id"]
        return {"access": "granted", "user": user_data}
    ```

    <Warning>
      If your tier is insufficient for a facility, you'll see exactly how many points you need to gain access.
    </Warning>
  </Step>

  <Step title="Start Inspection">
    ### Capture Toilet Photo

    After authentication, click "Start Toilet Inspection" to trigger the camera and begin analysis.

    **Frontend Workflow** (from `app/templates/qr_access_scanner.html:435`):
    ```javascript
    async function startToiletInspection() {
      // Create workflow session
      const response = await fetch("/api/start-toilet-inspection", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          user_id: currentUser.id,
          toilet_id: 1
        })
      });

      const data = await response.json();
      currentSessionId = data.session_id;
      
      // Poll for inspection status
      pollInspectionStatus();
    }

    async function pollInspectionStatus() {
      const response = await fetch(`/api/workflow-status/${currentSessionId}`);
      const data = await response.json();

      switch (data.status) {
        case "analyzing":
          setInspectionStatus("AI analyzing image...", "fa-brain");
          setTimeout(pollInspectionStatus, 2000);
          break;
        case "completed":
          showResults(data.analysis_result);
          break;
      }
    }
    ```

    **Backend Orchestration** (from `app/main.py:358`):
    ```python
    @app.post("/api/start-toilet-inspection")
    async def start_toilet_inspection(request: StartInspectionRequest):
        # Create workflow session
        session = WorkflowSession(
            user_id=request.user_id, 
            toilet_id=request.toilet_id
        )
        workflow_sessions[session.session_id] = session
        
        # Send session ID to camera clients via WebSocket
        for ws in connected_clients:
            await ws.send_text(f"SESSION:{session.session_id}")
        
        # Trigger camera capture
        for ws in connected_clients:
            await ws.send_text("CAPTURE")
        
        return {
            "status": "inspection_started",
            "session_id": session.session_id
        }
    ```

  <Step title="AI Analysis">
    ### Understanding the Vision System

    FlushLa supports three AI vision providers. The analysis detects issues and calculates cleanliness scores.

    **Vision Analysis** (from `app/vision.py:242`):
    ```python
    def analyze_toilet_image(image_bytes: bytes) -> dict:
        """Route to configured AI provider"""
        try:
            if VISION_PROVIDER == "openai":
                return analyze_with_openai(image_bytes)
            elif VISION_PROVIDER == "vertex_ai":
                return analyze_with_vertex_ai(image_bytes)
            elif VISION_PROVIDER == "cloud_vision":
                return analyze_with_cloud_vision(image_bytes)
        except Exception as e:
            # Fallback to default score
            return {
                "detections": [],
                "cleanliness_score": 75,
                "issues": [],
                "confidence": 0.5
            }
    ```

    **OpenAI GPT-4 Vision** (from `app/vision.py:116`):
    ```python
    def analyze_with_openai(image_bytes: bytes) -> dict:
        image_base64 = base64.b64encode(image_bytes).decode("utf-8")

        response = openai_client.chat.completions.create(
            model="gpt-4o",
            messages=[
                {"role": "system", "content": SYSTEM_PROMPT},
                {
                    "role": "user",
                    "content": [
                        {"type": "text", "text": "Analyze toilet cleanliness."},
                        {
                            "type": "image_url",
                            "image_url": {
                                "url": f"data:image/jpeg;base64,{image_base64}"
                            }
                        }
                    ]
                }
            ],
            temperature=0,
            max_tokens=1000
        )

        return json_safe_parse(response.choices[0].message.content)
    ```

  <Step title="Receive Results & Points">
    ### Score Calculation

    After analysis, your personal score is updated based on the toilet's cleanliness.

    **Score Delta Calculation** (from `app/main.py:228`):
    ```python
    def calculate_score_delta(cleanliness_score: int) -> int:
        """Calculate score delta based on cleanliness score"""
        if cleanliness_score >= 90:
            return 5  # Bonus for very clean
        elif cleanliness_score >= 70:
            return 0  # Neutral
        elif cleanliness_score >= 50:
            return -5  # Small penalty
        elif cleanliness_score >= 30:
            return -10  # Medium penalty
        else:
            return -15  # Large penalty for very dirty
    ```

    **User Score Update** (from `app/main.py:274`):
    ```python
    def update_user_score(user_id: str):
        # Get latest visit score delta
        res = supabase.table("toilet_visits")
            .select("score_delta")
            .eq("user_id", user_id)
            .order("created_at", desc=True)
            .limit(1).single().execute()
        
        score_delta = res.data.get("score_delta", 0)
        
        # Update user's total score (clamped 0-100)
        current_score = user_data.get("score", 0)
        new_score = max(0, min(100, current_score + score_delta))
        
        supabase.table("users")
            .update({"score": int(new_score)})
            .eq("id", user_id).execute()
        
        # Update Apple Wallet pass with new score
        update_wallet_pass(user_id, new_score)
    ```

    **Wallet Update** (from `app/main.py:297-334`):

    ```python
    # Update pass via PassCreator API
    url = f"https://app.passcreator.com/api/v3/pass/{identifier}"

    # Send push notification
    ```

  <Step title="Tier Progression">
    ### Understanding Tiers

    Your accumulated score determines your tier level and facility access.

    **Tier Calculation** (from `app/main.py:241`):
    ```python
    def get_user_tier(score: int) -> int:
        """Calculate user tier based on score
        Returns: 1 (Bronze), 2 (Silver), or 3 (Gold)
        """
        if score >= 80:
            return 3  # Gold
        elif score >= 50:
            return 2  # Silver
        else:
            return 1  # Bronze
    ```

    **Tier Names**:
    - **Tier 1 (Bronze Standard)**: 0-49 points - Basic access
    - **Tier 2 (Silver Standard)**: 50-79 points - Standard facilities
    - **Tier 3 (Gold Standard)**: 80-100 points - Premium facilities
